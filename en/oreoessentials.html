Highlights

A quick view of what OreoEssentials adds on top of a classic Essentials plugin: cross-server capabilities, database backends, and quality-of-life systems.

Core features

  • Homes / Warps / Spawn with cross-server teleport handoff (requestId + per-server queues).
  • Random Teleport (/rtp) with rank-based radius and fully configurable per-rank settings.
  • Chat tools with formatting, gradients and PlaceholderAPI support.
  • Portals & JumpPads for hub / lobby / adventure servers.
  • Moderation: ban, kick, mute, freeze and more.
  • QoL utilities: god/fly/vanish, warps, tpa, back, kits, and more classic essentials commands.
SmartInvs-based GUIs are used for many systems (menus, inventories, selectorsâ€¦).

Network features & storage

  • Cross-server economy with Vault support and MongoDB/PostgreSQL/JSON backends.
  • Cross-server EnderChest with rank-based size, locked slots & per-world rules.
  • Player sync (inventory / XP / health / hunger) with per-player toggles and global enable.
  • RabbitMQ bridge for cross-server signals plus proxy messaging for server switches.
  • Redis cache as an optional layer for high-traffic networks (less DB pressure).
  • Async operations everywhere possible to keep the main thread clean.
Designed for Velocity & BungeeCord networks, but works fine on single servers as well.
